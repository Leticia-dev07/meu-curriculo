# 📄 Currículo em HTML e CSS

Este projeto consiste em um currículo desenvolvido utilizando **HTML5** e **CSS3**, com o objetivo de praticar a estruturação de páginas web e a estilização básica com CSS.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Git
- GitHub

---

## 📚 O que foi praticado neste projeto

- Estrutura semântica com `header`, `main`, `section` e `footer`
- Organização do conteúdo em seções
- Estilização utilizando CSS
- Centralização do layout
- Criação do efeito de "folha" com `box-shadow`
- Estilização de links com efeito `hover`

---

## 🎯 Objetivo

Praticar os fundamentos do **desenvolvimento front-end**, entendendo como estruturar e estilizar um documento HTML de forma organizada, semântica e visualmente agradável.
